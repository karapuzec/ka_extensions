How to Install "Ka Extensions"
==============================

"Ka Extensions" library is provided as a standard OpenCart module.  
It is distributed as a ZIP archive with a name similar to:

::

    ka_extensions_ver4.1.1.24.ocmod.zip

The library is free and can be downloaded from the official OpenCart website
or from our official website.

Features
--------

- Does not overwrite any core files
- Fully compatible with OCMOD and VQMOD modifications

Installation Steps
------------------

1. Upload the archive via the OpenCart Extension Installer.
2. After a successful upload, go to:

   **Extensions → Modifications**

3. Click the **Refresh** button to rebuild the modifications cache.

Verification
------------

Once the cache is refreshed, you should see an ambulance icon at the top of the
admin panel.

This icon indicates the status of *kamod* modifications:

- If no issues are present, everything is working correctly.
- If errors occur, the icon will help you identify problems in modifications.
